# 🚀 Portfólio – Felipe Januario Alves

Portfólio profissional desenvolvido com HTML, CSS e JavaScript, com design moderno, responsivo e interativo. Exibe minhas principais habilidades, projetos, seções sobre mim e integrações com envio de formulários via backend (Node.js + Nodemailer).

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** (mobile-first, Flexbox, Grid)
- **JavaScript puro** (menu hamburguer, modais, fetch, animações)
- **AOS** – animações ao rolar a página
- **Backend com Node.js + Express + Nodemailer** (em desenvolvimento)
- **Formcarry** (gateway de formulário como alternativa ao backend)
- **Hospedagem**: domínio próprio `www.felipejanuarioalves.com.br` (Render / Railway / Vercel)

🚀 Meu Portfólio Profissional Está Online — e em Evolução Contínua
🔗 www.felipejanuarioalves.com.br

Depois de meses estudando, testando, errando, ajustando e codando com propósito, compartilho com orgulho o lançamento do meu site profissional. Mais do que um portfólio, esse projeto é uma vitrine da minha jornada como desenvolvedor fullstack, refletindo minha identidade, habilidades e visão de futuro.

💻 Sobre a Stack e as Escolhas Técnicas
A versão atual foi construída com HTML, CSS e JavaScript puro, respeitando boas práticas de acessibilidade, semântica, modularização e responsividade. O código foi pensado para crescer, com estrutura pronta para migração futura para React ou Next.js, e integração com um backend real em Node.js + Express.

🧠 Aplicando na prática o que aprendi no DevClub
Esse projeto é também um reflexo direto de tudo que venho aprendendo na formação fullstack do DevClub (Rodolfo Mori). Nele, apliquei:

Princípios de componentização de código e responsividade mobile-first;

Técnicas de manipulação de DOM com JS limpo;

Organização de pastas e arquivos para projetos reais;

Estruturação de código pensando em manutenibilidade e reaproveitamento;

Conceitos de clean code e atenção à experiência do usuário.

Tudo que antes era aula, agora virou prática. E isso faz toda a diferença.

🧩 Decisões estratégicas e adaptações reais
Optei por utilizar o Formcarry no lugar de uma API própria para envio de formulários — apesar do backend já estar desenvolvido. Isso se deu por uma questão financeira: o custo de hospedagem de servidores ainda é inviável no momento. Mas essa é uma questão temporária. Em breve, essa parte será atualizada com:

Deploy completo do backend com CI/CD;

Painel de administração para cadastrar novos projetos direto pelo navegador.

🔄 Melhorias em andamento e novidades planejadas
Esse projeto está em constante evolução. Algumas atualizações que já estão no roadmap:

✅ Layout repaginado com cores neutras e destaques visuais, priorizando contraste, legibilidade e estética moderna;

✅ Responsividade total em todas as seções;

🔜 Seção de skills visuais com gráficos interativos;

🔜 Mapa mental ilustrando minha trajetória como dev — da transição de carreira à atuação fullstack;

🔜 Backend funcional com formulário de cadastro de projetos e upload via painel próprio.

🔥 O que esse projeto representa
Esse portfólio não é só código — é a minha jornada viva em bytes. Carrega cada hora de estudo, cada aula prática, cada bug que virou aprendizado. Representa não só o que eu sou como dev hoje, mas onde quero chegar.

Se você curte tecnologia, design, carreira tech ou simplesmente admira projetos com propósito, acessa, navega e manda aquele feedback sincero.

Tamo só começando.
